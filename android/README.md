# Android Tools

Android applications built with Gradle on a Linux or macOS machine.

## Tools

| Tool | Description |
|------|-------------|
| [book-reader](book-reader/) | OCR + TTS book reader app (French/English) — [pre-built APK](book-reader/app-debug.apk) |
| [trilingua](trilingua/) | Fully offline speech translator for English/French/Hungarian (6 directed pairs) — [pre-built APK (~881 MB)](https://github.com/Gofer59/tools/releases/download/trilingua-v1.0/app-debug.apk) |

## Prerequisites

- Android SDK or Android Studio
- JDK 17+
