# Windows Tools

Desktop utilities compatible with Windows 10/11.

## Tools

| Tool | Description |
|------|-------------|
| [deck-reader](deck-reader/) | Multi-hotkey screen OCR + TTS for visual novels and reading |
| [threshold-filter](threshold-filter/) | Live screen binary threshold filter for OCR preprocessing |

## Prerequisites

- [Rust toolchain](https://rustup.rs) — install via `rustup-init.exe`
- [Python 3.10+](https://www.python.org/downloads/) — required for deck-reader
- [winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/) — required for deck-reader (included in Windows 10 1809+)
