# Outils Windows

Utilitaires de bureau compatibles avec Windows 10/11.

## Outils

| Outil | Description |
|-------|-------------|
| [deck-reader](deck-reader/) | OCR d'ecran + synthese vocale multi-raccourcis pour romans visuels et lecture |
| [threshold-filter](threshold-filter/) | Filtre de seuil binaire en temps reel pour le pretraitement OCR |

## Prerequis

- [Chaine d'outils Rust](https://rustup.rs) — installer via `rustup-init.exe`
- [Python 3.10+](https://www.python.org/downloads/) — requis pour deck-reader
- [winget](https://learn.microsoft.com/en-us/windows/package-manager/winget/) — requis pour deck-reader (inclus dans Windows 10 1809+)
